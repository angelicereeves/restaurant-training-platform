# Restaurant Training Platform

Multi-tenant, role-based training platform for restaurants.

## Features
- Multi-restaurant support via dynamic routing
- Role-based training paths
- Modules and lessons system
- Interactive checklists (coming next)
- Firebase authentication (planned)

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Firebase
- Vercel

npm install
npm run dev
